## 🔮 What is SpookFi?

SpookFi isn't just another trading bot. It's an **autonomous, intelligent brain** that monitors the entire market universe, trains its own machine learning models, and manages its own "Ghost Bags" without asking for your permission. Built with a sleek, playful Gen-Z Neo-Brutalist UI.

### 🧠 The Brain (Zero Config)
- **Universe Hunter:** Scans the Indian Equity Market and Global Crypto Markets to find the most volatile assets.
- **The Forge:** Automatically trains XGBoost models in the background using advanced "Alchemist Math" (Kalman Filters, Hurst Exponents, Volatility Bands) + Alternative Sentiment Data.
- **Risk Manager:** Manages stop-losses, position sizing, and takes profit automatically based on current market volatility.

## 🚀 Getting Started

Want to run the engine yourself? It's easy.

1. **Clone the repo**
   ```bash
   git clone https://github.com/mithun0524/SpookFi.git
   cd SpookFi
   ```

2. **Setup the Environment**
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. **Ignite the Engine**
   ```bash
   ./setup_and_run.sh
   ```
   *Then pop open `http://localhost:8000` to access the Observation Deck.*
